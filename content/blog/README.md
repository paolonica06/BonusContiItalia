# Blog

Questa cartella contiene le bozze degli articoli generate automaticamente.

## Struttura

- `drafts/`
  Bozze `.md` pronte da rivedere o pubblicare.

## Modalita operative

- Senza `OPENAI_API_KEY`
  Gli articoli vengono creati con un template strutturato e dati reali delle offerte.

- Con `OPENAI_API_KEY`
  Gli articoli possono essere generati con testo piu naturale tramite OpenAI Responses API.
